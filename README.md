# LSTI2307v2 - Project Restructuring

This project is a restructured version of the original LSTI2307 repository. It is organized into multiple branches to separate different activities and project milestones as per instructor requirements.

## Branch Structure

- **main**: Contains only this README file.
- **avanceProyectoFinal**: Contains the progress reports for the final project.
- **proyectoFinal**: Dedicated to the final version of the project.
- **actividad1 - actividad5**: Individual branches for class activities.
- **ejerciciosPractica**: Contains all practice exercises.

## Purpose
The purpose of this project is to maintain a clean and organized history of class activities, exercises, and the final project development.
